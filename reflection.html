Responsive design is a Graphic user interface (GUI) design approach used to adjust the content to the size of the various screens. So, that they can adopt the size of the browser space to guarantee stability with various devices. It uses various techniques to adapt the web-page layout to the viewing environment.

Responsive web design is becoming more important nowadays as majority of mobile users visit web site more frequently. Responsive web design works through Cascading Style Sheets (CSS), by using various settings to serve different style properties depending on some characteristics of the current device in use. It is highly in use as it is making the life of business owners, and advertisers easier as it is making it possible to access any website on any device.

Responsive web design uses breakpoints to determine the layout of the web site. These breakpoints are based on the width of the browser. In responsive design, the elements of the page reshuffle as the viewpoint change.

I learned that building web apps by using the concept of responsive design is relative easer as we need not to make different web apps for different devices, as responsive design make it sure that the web apps work across all devices by changing the layout corresponding to the device used.
